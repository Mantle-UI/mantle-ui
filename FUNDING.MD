# Funding Mantle UI

Mantle UI is an independent, community-maintained open-source project.

Financial support helps us maintain the project and cover costs such as:

- Domain and website infrastructure
- Documentation hosting
- Continuous integration services
- Development and maintenance work
- Community contributor rewards and funded tasks

## Sponsors

Organizations and individuals can support Mantle UI through:

- Open Collective: <link>

## How Funds Are Used

Project funds are used for Mantle UI-related expenses.

Examples:
- Infrastructure costs
- Maintenance work
- Bug fixes and improvements
- Documentation improvements
- Contributor payments for approved work

## Sponsored Development

Sponsors may help fund specific improvements or maintenance tasks.

Sponsored work:
- remains open source
- follows the same review process as normal contributions
- does not guarantee acceptance of a feature request
- does not give sponsors control over project direction

## Transparency

We aim to keep project funding transparent and use funds only for the benefit of Mantle UI and its community.
